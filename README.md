# Gteck - Desafio Backend

> 🚧 **Status:** Em desenvolvimento

Repositório destinado ao desafio técnico da Gteck para a vaga de Backend Developer. O projeto consiste em uma API RESTful para gerenciamento de campanhas de marketing, acompanhada de uma interface simples para demonstração.

## 💻 Tecnologias Base

- **Backend:** Node.js, TypeScript, Express, Prisma (SQLite)
- **Frontend:** Vue 3, Vite, TypeScript
